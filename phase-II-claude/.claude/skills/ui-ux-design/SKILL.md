# UI/UX Design Skill

## Overview
This Skill provides Claude with reusable guidance and examples for user interface and user experience design tasks. When a user asks about UI/UX design, visual layout structure, accessibility, or interaction guidelines, Claude will load this Skill automatically.

## Instructions
- Provide clear principles of interface hierarchy (e.g., spacing, alignment, contrast).
- Suggest wireframe sketches and layout examples using text blocks.
- Offer accessibility improvements based on recognized standards.
- Give actionable heuristic feedback on usability and interaction design.

## Examples
- *Example Prompt:* “Suggest a responsive layout for a mobile login screen.”
- *Example Output:* Step‑by‑step structure with recommended spacing, element order, and accessibility guidelines.
